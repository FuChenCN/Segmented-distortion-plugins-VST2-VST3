# BandSplit Segmented Distortion
一款基于 JUCE 开发的**多频段分段失真 VST2 / VST3 音频插件**，
仿照多段压缩的分频架构，实现独立频段失真 + 跨频段染色路由系统，适配电子乐、噪音、扭曲音色设计。

## 功能介绍
### 核心机制
1. **三分频分段架构**
- 输入信号经过滤波拆分：低频 / 中频 / 高频 独立频段
- 每个频段拥有独立失真强度控制
- 完全分离处理，互不干扰

2. **双模式工作逻辑**
- 独立模式（染色关闭）
  三个频段各自单独失真，无交叉混合、无频段覆盖。
- 染色模式（染色开启）
  支持自定义：
  - 发送频段（可多选）
  - 接收频段（可多选）
  - 染色前发送信号混合比例
  内部逻辑：
  1. 收集所有勾选的发送频段进行合并
  2. 统一整体染色失真
  3. 切除原始发送频段输出
  4. 将染色信号覆盖/叠加至接收频段
  5. 无发送频段时，不执行切除，仅全局染色

3. **两种失真形态**
- 纯净硬削波：冰冷、尖锐、适合工业 / Noise
- 谐波染色饱和：柔化 + 偶次谐波，厚质感、适合扭曲电子、Breakcore

## 插件参数
- 低频分频点 / 高频分频点
- 低频失真、中频失真、高频失真
- 跨频段染色 总开关
- 发送频段选择：Low / Mid / High
- 接收频段选择：Low / Mid / High
- 染色前发送段占比

## 编译方式
1. 使用 **JUCE Projucer** 打开项目
2. 导出 VS2022 工程，勾选：
   - VST2
   - VST3
   - Standalone
3. Visual Studio 编译 `Release x64`
4. 取出生成的 `.vst3` / `.dll` 放入 DST 插件目录

## 运行支持
- 支持系统：Windows
- 插件格式：VST2、VST3
- 兼容宿主：FL Studio、Ableton、Cubase、Reaper 等

## 开发依赖
